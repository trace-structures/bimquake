BIMQuake is a seismic resilience analysis module using the extraction of structural data from BIM (Building Information Model) objects.

- Website: [https://buildchain.ilab.sztaki.hu/](https://buildchain.ilab.sztaki.hu/)
- Documentation: *TO BE ADDED*
- Source code: [https://github.com/trace-structures/bimquake](https://github.com/trace-structures/bimquake)
- Bug reports: [https://github.com/trace-structures/bimquake/issues](https://github.com/trace-structures/bimquake/issues)

It provides:

- handling, visualizing and extracting geometric and material infromation of building data using IFC files
- linear and non-linear seismic resilience analysis of a given building
